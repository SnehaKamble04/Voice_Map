# 🎙️ Voice_Map – AI Voice-Based Navigation System

**Voice_Map** is an AI-powered voice recognition web application that allows users to search and navigate maps using voice commands. This project is built to enhance accessibility and provide a hands-free, intuitive way to interact with maps, powered by speech.

---

## 🔍 Features

- 🎤 **Voice Input for Location Search**  
  Users can speak the name of a location, and the app will recognize it using speech recognition.

- 🗺️ **Google Maps Integration**  
  View your searched location in an embedded interactive map.

- 💬 **Real-Time Speech Recognition**  
  Uses Web Speech API to capture and process voice input instantly.

- 🧑‍💻 **User-Friendly Interface**  
  Simple and clean UI that ensures ease of use across all devices.

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript  
- **Backend**: Python (Flask)  
- **APIs & Libraries**:
  - Google Maps JavaScript API
  - Web Speech API (built-in browser support)
  - Flask for backend routing

---

## 🚀 Getting Started

Follow these steps to run the project locally:

### 1. Clone the Repository

```bash
git clone https://github.com/SnehaKamble04/Voice_Map.git
cd Voice_Map
```

### 2. Install Dependencies

Ensure Python 3 is installed. Then run:

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
python app.py
```

### 4. Open the Application

Visit your browser at:

```
http://localhost:5000
```
